# Hwange Wildlife Management System

## Overview
A VB .NET + MS Access desktop application for managing wildlife records, anti-poaching patrols, and visitor bookings in Hwange National Park.

## Features
- **Wildlife Records**: Track animal sightings, species, and population data
- **Anti-Poaching Management**: Log patrol routes, incidents, and ranger reports  
- **Visitor Booking**: Manage tourist bookings and park entry permits
- **Reporting**: Generate SDLC documentation and management reports

## Technology Stack
- **Frontend**: VB .NET
- **Database**: MS Access
- **Documentation**: SDLC
